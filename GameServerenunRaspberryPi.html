3- SSH Setup

Per generar la clau he utilitzat la comanda  ssh-keygen -t rsa dins una màquina virtual.

Agregam la clau virtual dins el servidor de la Raspberry

        

Copiar la clau pública al fitxer authorized_keys del servidor SSH

Inhabilitar l’accés SSH per password fent només possible l’accés per clau pública.

Documentació per utilitzar SSH Agent:

El SSH Agent ens permet conectar-nos de forma segura i còmoda mitjaçant l’ús de claus privades.
El que farem serà que el servei es inici automàticament cada vegada que encenem la nostra màquina de windows.

Obrim el powershell i escrivim la comanda que farà que el servei de SSH agent funcioni i que es inici cada vegada automàticament.

Ara escriurem la comanda perquè el servei fagi start

I finalment comprovam que estigui encès.
